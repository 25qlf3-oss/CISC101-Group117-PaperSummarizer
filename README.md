# Research Paper Summarizer Project

This repository contains a modular AI system prompt architecture for summarizing academic research papers.

## Files

### system_prompt.md
The full system prompt used to instruct the LLM how to summarize papers safely and consistently.

### /modules
Each module controls part of the internal architecture:

| Module | Description |
|--------|-------------|
| 01 | Intake & setup |
| 02 | Section-by-section summarization |
| 03 | Guardrails and hallucination controls |
| 04 | Final rendering and formatting |
| 05 | Citation extractor |
| 06 | Key contributions summarizer |

## Features
- Multi-level summaries
- Hallucination mitigation
- Strict evidence logic
- Expert & lay summaries
- Mini glossary
- Modular structure inspired by Travel Planner
